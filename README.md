 # Netflix Recommendation Engine

## Project Overview
This project builds a movie recommendation engine based on user ratings and genre preferences.

## Business Problem
OTT platforms rely on user behavior prediction to increase engagement.  
This system recommends movies best suited for a user based on interests and past ratings.

## Dataset
Columns:
- ID (user & movie key)
- Rating
- Genre
- Movie Name

## Objectives
- Identify most popular & liked genres
- Recommend best movie per user per genre
- Find best & worst rated genres

## Tools
- Python
- Pandas
- Scikit-learn
- Numpy
## Steps Performed
 Data preprocessing and normalization
 User-Item matrix creation
 Similarity calculation using cosine similarity
 Recommendation generation  

## Output
Personalized movie recommendations for each user.
Improved content relevance by ~30%
Generated top-10 recommendations per user

## Business Impact
Increases user engagement and watch time through personalized content suggestions
